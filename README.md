# SafeSight – Hack the Galaxy 2026

SafeSight is a web application that provides students and women with a real-time safety companion for navigating Vancouver and Burnaby.

Users simply enter their current location and destination, and SafeSight pulls real crime data from Vancouver’s Open Data Portal. It then uses Gemini to translate the data into a calm, clear, plain-English safety briefing.

No overwhelming statistics, just a simple explanation of what has been happening nearby, what to be aware of, and the resources available if help is needed.
